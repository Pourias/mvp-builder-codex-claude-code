# Security Policy

## Supported Versions

This is an early-stage project. Security and bug fixes are best-effort, with the
active development line being:

| Version | Supported |
| --- | --- |
| `main` | Yes |

## Reporting A Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead:

1. Contact the maintainer privately through the repository owner's public GitHub profile contact path:
   https://github.com/Pourias
2. If GitHub private vulnerability reporting is enabled for this repository, prefer that channel.

When reporting a vulnerability, please include:

- a description of the issue
- affected files or workflow areas
- steps to reproduce, if known
- possible impact
- any suggested mitigation

## Response Expectations

This project is currently maintained on a best-effort basis.

The goal is to:

- acknowledge a credible report within 7 days
- assess impact and reproduction details
- fix or mitigate confirmed issues as reasonably quickly as possible

If the issue is not a security vulnerability, please use the normal support or issue channels instead.
