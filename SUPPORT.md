# Support

Thanks for checking out `mvp-builder`.

## Where To Get Help

- Bug report: open a GitHub issue with the bug report form
- Feature idea: open a GitHub issue with the feature request form
- Usage question: open an issue and label it as a question if a dedicated discussion space is not available yet
- Security issue: do not open a public issue; follow [SECURITY.md](SECURITY.md)

## Before Opening An Issue

- Read [README.md](README.md)
- Read [CONTRIBUTING.md](CONTRIBUTING.md)
- Check whether the same topic is already covered by an open or closed issue
- If your question is really about a prompt or state transition, include the relevant run artifact or exact state name

## Helpful Context To Include

- what you were trying to build
- what command you ran
- the current state name
- the relevant artifact or error output
- what you expected to happen
- what happened instead

The more concrete the report, the easier it is to help quickly.
